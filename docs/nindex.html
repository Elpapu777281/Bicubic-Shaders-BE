Bicubic Shader BE (BSBE)

A Shader/Renderdragon Materials for Minecraft: Bedrock Edition that aims to be beautiful look with colorful tones and added some visual effects/feature


Main Features
  • Sky-Void Colouring
  • World/Terrain Colouring
  • Block Light Source Colouring
  • Custom SunMoon
  • Custom Fog
  • Fake 3D Clouds
  • Water Bump/Normal Map
  • Sky and Cloud Reflection on Water
  • Underwater Caustic
