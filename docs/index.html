Bicubic Shaders BE

A Shader/Renderdragon Materials for Minecraft: Bedrock Edition that aims to be beautiful look with colorful tones and some features like underwater caustic, fake 3D cloud, sky and cloud reflection, waterbump, etc.




Changelog

  • new this shader support MCBE with Renderdragon engine
  • new style tonemap
  • improve clouds
  • improve sky
  • improve water reflection
  • more vibrant color
  • change underwater fog
  • added foggy effect
  • added Minecraft vanilla AO
  • remove sun reflection on water
  • little optimizations
